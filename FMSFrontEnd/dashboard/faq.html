Left Panel

A. Dashboard

The  Dashboard is the main page of the system. It allows the users to see the vehicle status in real time.

B. Drivers

The  Drivers page allows the user to view, create and modify drivers.


Add a Driver

Click  Add Driver Button

Enter the necessary information on the right side of the Drivers page.

Click the “Submit” button.

To Edit a Driver

Select the driver from the table that needs to be modified.

Change the necessary information. Click the “Submit” button to update the driver information
To Delete a Driver


Select the driver from the table and click the “Delete” button On the confirmation dialog box, click “OK” to delete or “Cancel” to abort.
C. Vehicles

The  Vehicles page allows the user to view, create and modify vehicles.


Add a Vehicle

Click  Add Vehicle beside Delete button

Enter the necessary information on the right side of the Vehicles page.

Configure the settings.
Click the “Submit” button.

To Edit a Vehicle

Select the vehicle from the table that needs to be modified.

Change the necessary information. Click the “Submit” button to update the vehicle information
To Delete a Vehicle


Select the vehicle from the table and click the “Delete” button On the confirmation dialog box, click “OK” to delete or “Cancel” to abort.
D. Zones
The Zones   page contains the list of all zones assigned under a company. It also allows creation and modification of the zones previously created.

To create a Zone

Enter Zone Name Select the type of zone Click the “Draw Boundary” button Select the shape of the boundary for the zone Draw the boundary for the zone on the map by double clicking to create points For polygons, create the points on all the corners of the boundary. For circles, create the center of the circle first then mark the edge of the circle to set the radius of the boundary.
Click “OK” to proceed with the zone creation or click “Close” to cancel

To Edit a Zone


Select the zone from the table that needs to be modified. Change the necessary information Click the “Submit” button to edit the zone.
To Delete a Zone


Select the zone from the table and click the “Delete” button On the confirmation dialog box, click “OK” to delete or “Cancel” to abort
E. Companies
The COMPANIES   page shows the list of companies and allows creation and modification of accounts.

F. Users
The USERS   page shows the list of user accounts under a company and allows creation and modification of accounts.

To Add an User Account

Click Add User at the Top Menu Bar Enter the necessary information on the right side of the Users page. Configure the settings Click the “Submit” button to add the user.
To Edit an User Account


Select the user from the table that needs to be modified. Change the necessary information Click the “Submit” button to edit the user.
To Delete an User Account


Select the user from the table and click the “Delete” button On the confirmation dialog box, click “OK” to delete or “Cancel” to abort
